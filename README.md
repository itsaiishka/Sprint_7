# Sprint_7 — API тестирование сервиса Яндекс Самокат

Проект содержит автоматизированные API-тесты для учебного сервиса **Яндекс Самокат**.  
Тесты написаны на Python с использованием **pytest**, **Allure** и **requests**.

---
## ⚙️ Требования

- Python 3.10+  
- Установленные зависимости (рекомендуется виртуальное окружение):

```bash
python -m venv .venv
source .venv/bin/activate   # Linux/Mac
.venv\Scripts\activate      # Windows
pip install -r requirements.txt

Пример requirements.txt:
pytest
requests
allure-pytest

Запуск всех тестов:
pytest -s -v

Запуск конкретного теста:
pytest -s -v Tests/test_courier_login.py
```

Содержимое тестов:

- TestCreateCourier — создание курьера: 
    Создание с валидными данными
    Создание с уже существующим логином
    Отсутствие бязательного поля (login или password)

- TestCourierLogin — авторизация курьера:
    Успешный логин
    Неверный пароль
    Несуществующий логин

- TestCreateOrder — создание заказа:
    Параметризация цветов: BLACK, GREY, оба, или без цвета
    Проверка наличия track в ответе

- TestOrderList — получение списка заказов:
    Проверка кода ответа
    Проверка наличия ключа orders и типа данных


-------------------------

🔹 Особенности
Все тесты независимы
Для курьеров используется фикстура, которая создаёт курьера перед тестом и удаляет после
Тесты можно запускать параллельно с использованием pytest-xdist (опционально)
Используется Allure для генерации красивых отчётов
